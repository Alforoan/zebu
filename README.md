## About The Project



![zebu](https://github.com/Alforoan/zebu/blob/main/zebu.png)
 

`github_username`: Alforoan

`repo_name`: zebu

`twitter`: [Click Here](https://twitter.com/Alforoan2050)

`project_title`: Zebu

`project_description`: 
- Uses Postgresql for database, React for frontend, and Node for backend
- User can create, rename, update, delete flashcards and decks that hold flashcards
- User can search decks and cards
- Utilizes spaced repetition for flashcards to enhance memory retention
- Utilizes JWT tokens (access and refresh tokens) for secure authentication and session management

   ```sh
   git clone https://github.com/Alforoan/FullStackApi.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

7. npm run dev
